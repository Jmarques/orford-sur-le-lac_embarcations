Modifier un modèle de courriel

Trois pistes pour ajuster le texte des courriels préparés, sans risquer de casser les informations que l'app ajoute pour vous.

Le modèle sert à préparer le courriel « Relance — emplacement libre ». Comme toujours, rien n'est envoyé automatiquement : vous relisez chaque courriel avant de l'envoyer.

Piste A — Le texte avec des jetons { }

Tout le texte est libre. Les informations ajoutées par l'app s'écrivent entre accolades — les boutons les insèrent pour vous.

Insérer une information à l'endroit du curseur :
Enregistrer le modèle Revenir au texte d'origine

Aperçu — ce que recevra Marie Gagnon

L'aperçu se met à jour pendant que vous écrivez.

Piste B — Vos paragraphes autour d'informations verrouillées

Vous écrivez vos paragraphes ; les blocs marqués d'un cadenas sont remplis automatiquement par l'app — impossible de les casser, mais ils ne peuvent pas se glisser au milieu de vos phrases.

Objet du courriel
Votre emplacement 14 — Orford sur le Lac
Salutation
Bonjour Marie Gagnon,
Les détails de l'emplacement
Emplacement 14 · attribué au 234, rue du Pré · libre depuis le 12 juin

Piste C — Les informations comme des puces dans le texte

Le texte est libre, et chaque information de l'app est une puce insécable : elle se déplace ou se retire d'un bloc, jamais lettre par lettre. Les boutons remettent une puce retirée.
